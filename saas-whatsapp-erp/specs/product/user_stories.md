# User Stories & Use Cases

## 1. Authentication
- **UC-01 (Login):** As a user, I want to log in with my email and password so I can access my company's data.

## 2. Configuration
- **UC-02 (Settings):** As an admin, I want to configure the company tax rate and invoice sequence so I can correctly generate fiscal documents.

## 3. Customers
- **UC-03 (Management):** As a seller, I want to create or update customer profiles including their WhatsApp number so I can communicate with them.
- **UC-04 (History):** As a seller, I want to view a customer's past sales and invoices so I can provide better support.

## 4. Products
- **UC-05 (Catalog):** As an admin, I want to manage products and services so they are available for the POS.

## 5. Sales & POS
- **UC-07 (Quick Sale):** As a seller, I want to create a sale from a WhatsApp conversation so I don't have to switch contexts.
- **UC-08 (Calculation):** As a seller, I want the system to automatically apply taxes and discounts so I don't make mistakes in totals.

## 6. Invoicing
- **UC-09 (Generation):** As a user, I want to generate an invoice from a completed sale so I can formalize the transaction.
- **UC-10 (Delivery):** As a user, I want to send the invoice PDF via WhatsApp so the customer receives it instantly.
- **UC-11 (Payment):** As a user, I want to mark an invoice as paid so I can track my revenue.

## 7. WhatsApp Core
- **UC-12 (Chat):** As a seller, I want to see all my WhatsApp conversations in one place so I can respond quickly.

## 8. Usage Limits
- **UC-14 (Validation):** As the system, I want to check plan limits before allowing message sending or invoice generation to ensure plan integrity.
