# UI SCREENS – CUSTOMERS

## Screen: Customer Detail

### Propósito
Mostrar contexto comercial del cliente.

### Contenido
- Nombre
- Teléfono (WhatsApp)
- Última conversación
- Última venta
- Estado de pago

### Acciones
- Ver conversación
- Nueva venta
- Enviar recordatorio

### Restricción
No construir CRM pesado.
