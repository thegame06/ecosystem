# UI SCREENS – WHATSAPP MODULE (CORE)

## Screen: WhatsApp Home

### Propósito
Mostrar conversaciones y estado comercial del cliente.

### Contenido obligatorio
- Lista de conversaciones
- Último mensaje
- Nombre del cliente
- Estado comercial:
  - Nuevo lead
  - Venta creada
  - Factura enviada
  - Pendiente de pago

### Acciones visibles
- Crear venta
- Generar factura
- Enviar factura por WhatsApp

### Restricciones
- No métricas
- No configuraciones
