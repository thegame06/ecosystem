# UI / UX PRINCIPLES – WHATSAPP FIRST SAAS

## Principio rector
Este producto es WhatsApp-first.
La UI debe hacer evidente en menos de 10 segundos que el sistema sirve para:
- Vender por WhatsApp
- Facturar desde WhatsApp
- Dar seguimiento a clientes por WhatsApp

## Prohibiciones
- No dashboards tradicionales
- No pantallas llenas de métricas
- No navegación tipo ERP clásico
- No flujos largos

## Regla de oro
Si el usuario no puede:
- Ver un cliente
- Crear una venta
- Generar una factura
- Enviar por WhatsApp

En menos de 3 clics, la UI es incorrecta.

## Jerarquía visual
1. Conversaciones
2. Acciones comerciales
3. Estados del cliente
4. Configuración (último)
