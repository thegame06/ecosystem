# UI SCREENS – BILLING

## Screen: Invoice View

### Propósito
Emitir y enviar factura sin fricción.

### Contenido obligatorio
- Datos del cliente
- Total
- Estado de factura

### Acciones principales
- Enviar por WhatsApp
- Descargar PDF
- Marcar como pagada
- Cancelar (nota de crédito básica)

### Regla
Enviar factura por WhatsApp debe ser una acción principal,
no secundaria.
