# UI SCREENS – SALES & POS

## Screen: Create Sale (Quick POS)

### Propósito
Crear una venta en menos de 30 segundos.

### Flujo mínimo
1. Cliente (auto-seleccionado desde chat)
2. Agregar productos / servicios
3. Ver totales (subtotal, IVA, total)
4. Guardar venta

### Restricciones
- No formularios largos
- No pasos innecesarios
- No navegación a otras pantallas
